# 📶 How to Create a Guest WiFi Network on TP-Link Router (Archer A6)

## 🧾 Overview
This guide explains how to configure a **Guest WiFi Network** on a **TP-Link Archer A6 router** to separate guest traffic from your main network, improving **security and privacy**.

---

## 🖥️ Background
- Device: **TP-Link Router | Archer A6**

---

## ⚙️ Step-by-Step Configuration

### 1. 🌐 Open Router Login Page
- Open **Google Chrome** (or any web browser).
- Enter the router IP address:
  
```

192.168.8.200

```

---

### 2. 🔐 Login to Router
- Enter the user credentials:

```
Username: admin
Password: ******
```

- Click **Login**

---

### 3. 🧭 Navigate to Guest Network Settings
- Go to the **Advanced** tab.
- In the **left navigation menu**, go to:

```

Wireless > Guest Network

```

---

### 4. 📡 Configure Guest Network

In the **Guest Network** section:

> This feature creates a **separate network** for guests to ensure **network security and privacy**.

#### Settings:
- ✅ Tick mark on: **2.4GHz**
- 📛 Network Name (SSID):  
```

My_Guest_Wifi

```
- 🔒 Security:  
```

WPA/WPA2-Personal

```
- 🔑 Password:  
```

****************

```

---

### 5. 👥 Guest Permissions

- ✅ Tick mark on:
```

Allow guest to access your local network

```

---

### 6. 💾 Save Configuration
- Click:
```

Save

```

---

## 📶 Verification

### 7. 🔍 Check WiFi Network on Laptop
- Go to your laptop’s WiFi settings.
- View available networks.

✅ You should see:
```

My_Guest_Wifi

```

---

### 8. 🔑 Connect to Guest Network
- Click on:
```

My_Guest_Wifi

```
- Enter the password you configured earlier.

---

## ✅ Result
- The guest network is now successfully created.
- Guests can access WiFi using the **separate network**, improving overall security.

---

## 🙏 Thank You
Your guest WiFi network is now ready for use.
